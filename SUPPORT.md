# Support MyDacia (public)

## Avant d'ouvrir un ticket

1. Verifier la configuration des identifiants MyDacia.
2. Verifier les logs Jeedom `mydacia`.
3. Verifier si la donnee est visible aussi dans l'application MyDacia.

## Ouvrir un ticket

Merci d'inclure :

- version Jeedom,
- version du plugin,
- modele du vehicule,
- etape reproduite,
- extrait de log pertinent (sans information sensible).
